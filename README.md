# Shape-Recogniser
Implement an algorithm in Matlab that recognises the shape of each objects in an picture.


## Classifiers used

- MDE (Minimum Euclidean Distance)
- MDM (Minimum Mahalanobis Distance)
- KNN (k-Nearest Neighbours)
